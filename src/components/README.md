# Reusable Primitives UI Components

Este diretório contém os componentes de interface (UI) puramente reutilizáveis, seguindo o padrão de design visual Stripe-like:
- **Card**: Cartões com espaçamento amplo, bordas sutis e sombras suaves.
- **Button**: Botões com micro-animações no clique e estados hover/active de alto padrão.
- **Table / List**: Tabelas arredondadas e de fácil leitura.
- **Modal / Form Inputs**: Inputs de texto, data e seletores com design intuitivo.
- **Charts (SVGs)**: Gráficos vetoriais simples para relatórios.
