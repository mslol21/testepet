# CMS & Configurações Module

Este módulo gerencia o painel administrativo de customização visual do pet shop:
- Edição de identidade visual (Paleta de cores, Logo, Favicon, Banner)
- Edição de conteúdos (Textos, Depoimentos, Perguntas Frequentes, Equipe)
- Definição de horários de funcionamento e canais de contato
- SEO (Tags, metatitle, metadescription, palavras-chave)
