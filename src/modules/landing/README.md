# Landing Page Module

Este módulo gerencia o conteúdo e os blocos de exibição visual da página pública do pet shop (Storefront):
- Seção Hero, Slogan, Banner Principal
- Grid de Serviços Oferecidos
- Galeria de Transformações (Antes/Depois)
- Equipe, Depoimentos, FAQ
- Bloco de Contatos, Mapa e Links de Redes Sociais
