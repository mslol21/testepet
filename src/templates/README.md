# Deployment Templates

Este diretório contém os templates de novas instâncias e configurações prontas para novos pet shops parceiros:
- Configurações padrão de cores (Paletas sugeridas: azul-calixto, esmeralda-mint, violeta-pet, etc.)
- Estruturas prontas de catálogo (Banho, Tosa Máquina, Tosa Tesoura, Consulta Veterinária)
- Horários operacionais padrão
